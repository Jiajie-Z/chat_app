# Todo Manager

Welcome to the Todo Manager! This is a full-stack web application that allows users to manage their tasks effectively. The app includes features such as user authentication, task management, due date tracking, and adding notes to tasks.

---

## **Features**
- **User Authentication**:
  - Login with a username to access your personalized task list.
  - Secure session management with cookies.
  
- **Task Management**:
  - Add, edit, and delete tasks.
  - Mark tasks as completed.
  - Add optional due dates and notes to each task.
  
- **Responsive Design**:
  - Mobile-friendly and desktop-friendly layout.
  - Scrollable task list for large lists.

- **Discoverable Functionality**:
  - Clear and intuitive interface for seamless user experience.

---

## **How to Use**
1. **Login**:
   - Enter a username on the login page.
   - Once logged in, you will see your personal task list.

2. **Add a Task**:
   - Use the input field to enter a task and an optional due date.
   - Click the "Add" button to add the task to your list.

3. **Manage Tasks**:
   - Mark a task as complete by checking the box next to it.
   - Add or edit notes for each task in the provided textarea.
   - Delete a task using the red delete button.

4. **Refresh or Logout**:
   - Use the "Refresh" button to reload your tasks.
   - Click "Logout" to exit your session.
